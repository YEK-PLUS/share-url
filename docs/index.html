share-url1.2.0

Create social sharing url strings

Generate dynamic social sharing urls without downloading its third party scripts. Customize for your UI needs. minified + gzipped.

npm i share-url -S

Usage Example

<a href={ShareUrl.facebook({ u: 'https://google.com' })}>
  Share on Facebook
</a>
const anchor = document.querySelector('a');
anchor.href = ShareUrl.facebook({ u: 'https://google.com' });

Check README for additional options.

ShareUrl object available on window for quick tests.
